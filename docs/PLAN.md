# Продуктовий план — AI Sport Assistant

## Бачення
Повноцінний особистий AI спортивний асистент: трекінг бігу/ходьби, контроль харчування через
комп'ютерний зір, і AI-коуч на базі Claude, який бачить твої дані, складає плани тренувань і дає
поради щодо здоров'я та харчування.

## Цільова платформа
iOS (нативно, Swift/SwiftUI), з компаньйоном для Apple Watch на пізнішій фазі.

---

## Модулі

### 1. Біг та ходьба (Фаза 1) 🏃
- Дозволи HealthKit / CoreMotion / CoreLocation
- Запис тренування: старт/пауза/стоп
- Живі метрики: час, дистанція, темп, пульс, кроки, калорії
- GPS-трек маршруту
- Історія тренувань + деталі забігу
- Локальне сховище (SwiftData)

### 2. AI-асистент (Фаза 2) 🤖
- Чат із Claude (tool use)
- Інструменти моделі: читання історії тренувань, профілю, харчування
- Генерація програми тренувань під ціль
- Відповіді на питання про харчування та здоров'я
- Щотижневий авто-звіт від Claude

### 3. Калорії по фото (Фаза 3) 📸
- Фото страви → калорії + БЖВ (білки/жири/вуглеводи)
- Claude vision як основний движок розпізнавання
- Історія харчування, денний баланс під ціль
- Штрихкод-сканер для упакованих продуктів

### 4. Здоров'я та відновлення (Фаза 4) ❤️
- Сон, HRV → щоденний Readiness score
- VO₂max, пульс спокою, тренди
- Гідратація
- Профілактика травм (контроль зростання навантаження, ACWR)
- Apple Watch app (живий пульс, керування тренуванням)
- Адаптивні плани (Claude переписує тренування за відновленням)

### 5. Додаткові ідеї (беклог) ✨
- Голосовий інтерфейс до асистента
- Аналіз техніки бігу по відео (Vision pose detection)
- Інтеграції: Strava, Bluetooth-ваги
- Рекорди (PR), сегменти
- Розумні пуш-нагадування з мотивацією
- Цілі: схуднення / марафон / набір форми

---

## Принципи
- **Приватність:** дані здоров'я максимально on-device; на бекенд — лише потрібний контекст для асистента.
- **Claude у центрі:** асистент через tool use отримує структурований доступ до даних, а не «сирий» дамп.
- **Інкрементальність:** кожна фаза дає робочий, корисний застосунок.
